Enable Icons Menus Trong Hệ Điều Hành Linux

2016-04-02

Contents

Mặc định trong Ubuntu và Fedora icon menu sẽ bị tắt đi. Dưới đây là cách bật icon menu trên 2 distro Linux là Ubuntu và Fedora: Để bật chế độ menu có icon, cần thực hiện lệnh sau:

1
2

#!/bin/bash
~$ gsettings set org.gnome.desktop.interface menus-have-icons true

hoặc:

1

~$ gsettings set org.gnome.settings-daemon.plugins.xsettings overrides "{'Gtk/ButtonImages': <1>, 'Gtk/MenuImages': <1>}"
